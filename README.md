# Simulador de Hidrômetro Analógico (SHA) - Versão 2.0

Este projeto implementa um simulador de hidrômetro em C++ utilizando Programação Orientada a Objetos.  
A versão 2.0 introduz a **simulação ininterrupta**, permitindo acompanhar a leitura do volume de água sem tempo definido de execução.

## Funcionalidades
- Simulação contínua (loop infinito).
- Exibição de leituras no terminal.
- Configuração de parâmetros via `config.txt`.
